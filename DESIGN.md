# Design System

## Scene

A judge opens a recovered 2126 archive node in a dim room and sees an operating-system artifact, not a normal portfolio page.

## Palette

Near-black and graphite carry the full surface. One dominant cyan-green signal color drives the archive UI, borders, status indicators, scan effects, and calls to action. Muted cool greys support copy.

## Typography

Use a clean system sans stack for high-impact display and body copy, plus a system mono stack for command, metadata, file labels, and terminal surfaces. This avoids external font fetches during deployment.

## Components

- Classified access hero with scan line, identity card, and status telemetry.
- Redacted operator file with abstract geometric mask.
- Capability modules with status indicators and hover states.
- Archived system records with codename, category, utility metadata, and signal meter.
- Interactive terminal with command chips, typed input, and signal effect.
- Compact timeline and final transmission.

## Motion

Use restrained scan, glow, and reveal motion. Respect reduced-motion preferences.
